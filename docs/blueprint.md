# **App Name**: Africast

## Core Features:

- Probabilistic Dashboard: Visualizes time-series forecasts using Interactive P10, P50, and P90 confidence intervals for risk-aware decision making.
- Regional Context Intelligence: A tool that leverages generative AI to adjust forecasts based on regional specifics like African holiday calendars and mobile network stability data.
- Financial Pattern Projector: Specialized forecasting for daily transaction throughput with context-aware triggers for month-end salary cycles.
- Linguistic Growth Analytics: Generative tool to predict language usage trends and term frequency shifts specifically for African dialects like Shona and Ndebele.
- Time-Series Data Layer: Structured storage and management system for transaction logs and historical API metrics to feed forecasting models.
- Anomaly & Drift Alerts: Automated system that notifies users of significant distribution shifts or forecasting errors beyond acceptable thresholds.
- Live Inference API Hub: A developer portal to manage and monitor REST endpoints that provide real-time /forecast and /intervals responses.

## Style Guidelines:

- A sophisticated dark mode palette inspired by 'Midnight Earth' and technical precision. Background is a deep charcoal with an orange undertone (#181513).
- Primary color: Vibrant Earth Orange (#EB8C47) for highlights and primary metrics, providing high contrast against the dark base.
- Accent color: Deep Rose (#E87E89), an analogous hue used for uncertainty bands and critical alerts.
- Font pairing: 'Space Grotesk' for headlines to evoke a scientific, computerized feel; 'Inter' for body text to maintain maximum legibility for dense data metrics.
- Minimalist, thin-stroke SVG icons that reflect technical infrastructure and geographic nodes, utilizing the accent Rose for active states.
- A high-density grid-based dashboard optimized for viewing large datasets, similar to an award-winning administrative tool.
- Staggered 'trace-line' animations for time-series charts when shifting between datasets to emphasize temporal flow.
